Public Cloud

  • Eine public Cloud ist eine Cloud, die öffentlich zugänglich ist.
  • Die Hauptvorteile der public Cloud sind folgende: Unternehmen müssen weniger in eigene IT-Ressourcen investieren und eigene IT-Ressourcen vor Ort zu investieren und zu unterhalten
  • Im Kern besteht eine public Cloud jedoch aus den folgenden Hauptmerkmalen: On-Demand-Computing und Self-Service-Bereitstellung. breiter Netzzugang. Ressourcen-Pooling.
  • On-Demand-Computing: die Fähigkeit, Rechenressourcen nach Bedarf bereitzustellen.
  • Self-Service-provisioning: die Möglichkeit, Rechenressourcen bereitzustellen, ohne dass ein einen Administrator, der den Bereitstellungsprozess verwaltet.

Allgemein Cloud

  • Die Cloud stellt Speicherplatz, Rechenleistung und ausführbare Software in einem räumlich entfernten Rechenzentrum zur Verfügung.
  • Was sind Beispiele für Cloud Computing? Die geläufigsten Cloud Computing Beispiele sind Apple iCloud, Google Drive, Microsoft OneDrive, Amazon Web Services und Open Telekom Cloud. Weitere interessante Anwendungen, wie Software-Anwendungen zur Zeiterfassung, Buchhaltung usw., werden als Cloud Dienste angeboten.
  • Cloud Computing beschreibt ein Modell, das bei Bedarf – meist über das Internet und geräteunabhängig – zeitnah und mit wenig Aufwand geteilte Computerressourcen als Dienstleistung, etwa in Form von Servern, Datenspeicher oder Applikationen, bereitstellt und nach Nutzung abrechnet.
